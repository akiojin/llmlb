# クイックスタート: SPEC-08d2b908: モデル管理（統合仕様）

## 前提条件

| 項目 | 要件 |
|------|------|
| ダッシュボード | 管理UIにログイン済み |
| エンドポイント | 1台以上登録済み |

## 基本的な使用例

### 1) エンドポイント登録

ダッシュボードの「Endpoints」からエンドポイントを追加します。

### 2) モデル一覧の同期

エンドポイント登録後、自動同期または手動同期を実行し、
`/v1/models` の一覧を更新します。

### 3) モデルの確認

ダッシュボードのモデル一覧で、各エンドポイントの対応モデルを確認します。
