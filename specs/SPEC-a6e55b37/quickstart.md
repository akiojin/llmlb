# クイックスタート: 自動アップデート（開発者向け）

## 1) 更新の検知
- 起動するとバックグラウンドで GitHub Releases を確認し、更新があればダッシュボードに表示される。

## 2) 更新の承認
- ダッシュボードの Update banner から `Restart to update` を押す。
- macOS/Windows はトレイメニューからも `Restart to update` を押せる。

## 3) ドレイン
- 承認後、新規推論 `/v1/*` は 503 になる。
- 処理中リクエストが完了したら更新が適用され、llmlb が再起動する。

## 4) 失敗時
- 権限不足などで自動適用できない場合は、ダッシュボードに手動手順が表示される。
