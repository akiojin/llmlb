# クイックスタート: Web UI 画面一覧

## 概要

LLM Load Balancer Web UIの画面構成と基本的な操作方法を説明する。

## 画面アクセス

### ログイン画面

```text
http://localhost:8080/dashboard/login.html
```

デフォルト認証情報（開発モード）:

- ユーザー名: `admin`
- パスワード: `test`

### ダッシュボード

```text
http://localhost:8080/dashboard
```

ログイン後に自動遷移。

### Playground

```text
http://localhost:8080/playground
```

ダッシュボードの「Chat」ボタンからアクセス。

## 画面遷移フロー

### 初回アクセス

1. `/dashboard` にアクセス
2. 未認証の場合、`/dashboard/login.html` にリダイレクト
3. 認証情報を入力してログイン
4. ダッシュボードが表示される

### Playgroundへの遷移

1. ダッシュボードのヘッダーで「Chat」ボタンをクリック
2. Playground画面が開く
3. 「Dashboard」ボタンでダッシュボードに戻る

## ダッシュボードセクション

### Stats Cards

システム全体の統計情報を表示。

- ノード数
- リクエスト数
- 稼働時間

### Node Table

登録されているノードの一覧を表示。

- ノード名、IP、ステータス
- GPU情報（モデル、能力スコア）
- 詳細モーダルで設定変更可能

### Request History

リクエスト履歴をページネーション付きで表示。

- モデル名、ステータス、時間
- 詳細モーダルで完全なリクエスト/レスポンスを確認

### Log Viewer

ロードバランサーのログをリアルタイム表示。

- ログレベルでフィルタリング
- 自動更新（10秒間隔）

### Models Section

登録されているモデルの一覧を表示。

- モデル名、タイプ、ノード割り当て
- 新規モデル登録モーダル

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Esc` | モーダルを閉じる |
| `Ctrl+/` | テーマ切替 |

## トラブルシューティング

### ログイン画面にリダイレクトされる

- JWTトークンの有効期限切れ
- ブラウザのCookieをクリアして再ログイン

### 画面が読み込まれない

- ロードバランサーが起動しているか確認
- ブラウザの開発者ツールでエラーを確認

### ノードが表示されない

- ノードがロードバランサーに登録されているか確認
- ノードのヘルスチェックが成功しているか確認
